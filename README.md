# SillyMoji (◕‿◕)

A kaomoji (顔文字) picker extension for [SillyTavern](https://github.com/SillyTavern/SillyTavern).

Quickly insert Japanese emoticons into your chat with a single click!

![Version](https://img.shields.io/badge/version-1.0.0-blue)

## Features

- **350+ kaomoji** across 15 categories
- **Quick access button** — sits on the right side of the chat input bar
- **Category tabs** — Happy, Sad, Angry, Love, Surprise, Greeting, Shrug, Animals, Table Flip, Sparkle, Lenny, Nervous, Dancing, Eating, Music
- **Search** — filter kaomoji by category name
- **Recent history** — remembers your last 20 used kaomoji
- **Smart insert** — inserts at cursor position with automatic spacing
- **Theme-aware** — adapts to your SillyTavern theme

## Installation

### Via SillyTavern UI (Recommended)

1. Open SillyTavern
2. Go to **Extensions** > **Install Extension**
3. Paste the repository URL and click **Install**

### Manual

1. Navigate to your SillyTavern installation directory
2. Go to `data/<user-handle>/extensions/third-party/`
3. Clone or copy this repository:
   ```bash
   git clone https://github.com/YourUsername/SillyMoji.git
   ```
4. Restart SillyTavern

## Usage

1. Look for the **ツ** button on the right side of your chat input bar
2. Click it to open the kaomoji picker
3. Browse categories using the icon tabs, or type to search
4. Click any kaomoji to insert it into your message
5. Your recently used kaomoji appear in the **clock** tab for quick re-use

Press **Escape** or click outside the picker to close it.
